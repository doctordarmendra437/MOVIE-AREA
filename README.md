# 🎬 Movie Area ⚠️

A Telegram Movie Bot that:

- ✅ Auto-indexes movies from your given channels  
- ✅ Stores them into MongoDB  
- ✅ Lets users search movies  
- ✅ Sends results with **shortened links (earnlinks.in)**
